# SmartLearn AI 🎓

A cloud-based smart learning platform designed to provide interactive and modern online learning experiences. The platform includes secure user authentication, course enrollment, quizzes, AI-powered assistance, and cloud storage integration using AWS services.

---

# 🚀 Features

- 🔐 User Authentication using AWS Cognito
- 📚 Course Enrollment System
- 📝 Interactive Quiz Module
- 🤖 AI Chatbot Assistant
- ☁️ AWS Cloud Integration
- 📄 PDF Lesson Viewer
- 📊 Student Dashboard
- 🎯 Progress Tracking
- 📱 Responsive User Interface

---

# ⚙️ How It Works

1. Users can register and log in securely using AWS Cognito authentication.
2. Students can enroll in different courses available on the platform.
3. Course lessons and study materials are stored and managed using AWS cloud services.
4. Users can attend quizzes and track their scores.
5. The AI chatbot helps students with guidance and learning support.
6. Student progress and enrollment data are managed using DynamoDB.
